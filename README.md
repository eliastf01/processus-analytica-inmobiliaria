# Processus Analytica Inmobiliaria

## Visión General
Esta plataforma es el brazo tecnológico de **Processus SRL**, diseñada para transformar 18 años de experiencia en evaluación de proyectos y estudios de mercado en una solución digital de alta precisión para el sector inmobiliario.

## Pilares del Proyecto
1. **Big Data Histórico:** Integración de bases de datos de mercado desde 2008.
2. **Evaluación de Proyectos:** Algoritmos propios para el cálculo de ROI, TIR y VAN en tiempo real.
3. **Gestión de Inventario:** Listado de propiedades validadas técnicamente por nuestra consultora.

## Tecnologías
- **Frontend:** Next.js (React)
- **Base de Datos:** PostgreSQL (Supabase)
- **Despliegue:** Vercel
